# slack-js-console
A JavaScript interpreter for Slack channels

# Installation

```
git clone https://github.com/xpl/slack-js-console
cd slack-js-console
npm install
```

# Configuration

Edit the `config.json` file:

```json
{
    "accessToken": "<YOUR ACCESS TOKEN HERE>",
    "channel": "javascript"
}
```

# Running

```
npm start
```
